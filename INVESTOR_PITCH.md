## The Problem

Every day, 165 million retail traders in the US make decisions with delayed, fragmented information. When Trump posts about tariffs, oil spikes 5% in pre-market — by the time a retail trader reads the headline on CNBC, the move is done.

Meanwhile, institutions have:
- Real-time political sentiment analysis
- Dark pool and options flow data
- Insider buying alerts
- Sector rotation tracking

**Retail traders pay the "information delay tax" on every trade.** The tools that solve this cost $100-250/month and are designed for professionals, not the 75% of retail traders who trade from their phones.

## Technology

### Architecture
```
Frontend (Next.js) → API Routes → Data Pipeline → Supabase (Auth + Postgres)
                                       ↓
                          ┌─────────────┼─────────────┐
                          ↓             ↓             ↓
                     RSS/News      SEC EDGAR      Market Data
                     Aggregator    (13F/Insider)   (Finnhub/Polygon)
                          ↓             ↓             ↓
                     Claude API    Ticker Parser   WebSocket
                     (Sentiment)   (Smart Money)   (Real-time)
```

### Key Technical Advantages
- **Server-side data fetching** eliminates CORS issues and API key exposure
- **WebSocket connections** for real-time price updates (no polling delay)
- **Edge functions** on Vercel for sub-100ms response times globally
- **LLM-powered sentiment** via Claude API for 95%+ accuracy vs 65% keyword-based
- **Scalable from day one** — Supabase + Vercel scale automatically

## Why Now?

1. **Political trading is a new asset class.** Trump's social media posts move markets instantly. No existing tool is purpose-built for this.

2. **Retail trading has exploded.** $302B retail equity inflows in 2025 (+53% YoY). 165M Americans now trade stocks.

3. **75% trade on mobile.** Most institutional-grade tools are still desktop-first. Mobile-first is a massive gap.

4. **AI enables real-time sentiment at scale.** LLMs like Claude can now analyze headlines with institutional accuracy at pennies per analysis. This wasn't possible 2 years ago.

5. **Smart money data is becoming democratized.** SEC EDGAR, FINRA, and options exchanges provide free/cheap data that was previously only accessible to institutions. Someone needs to package it for retail.
